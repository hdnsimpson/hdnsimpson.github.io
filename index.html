My Projects _

NavCoin Core Wallet Community Fund GUI

Using Qt and C++, I created an interface for the Community Fund in the NavCoin Core Wallet, previously only interactable via command line. This allowed users to create their own proposals and payment requests, as well as sort and vote for both.

Unreal Engine Perceptive AI

A perceptive Skeleton Warrior AI implemented in Unreal Engine 4 using the Behaviour Tree. The AI utilize a Vision, Hearing and Damage sensing perception system in order to react to stimuli to drive their Idle, Investigating and Attacking behaviours. All animation blueprints and montages as well as sound cues were implemented as part of this project too.

Crypto Cognizant

A full stack web app with a TypeScript React frontend, C# ASP.NET Core backend, SQL relational database and all hosted on Microsoft Azure. Crypto Cognizant leverages the CryptoCompare.com API to allow users to find exchanges to sell their coins on for the currencies they want.

COVID-19 Tracker

A React JS app utilizing a RESTful API and axios to provide tracking for statistics around the 2019 Coronavirus, hosted on Heroku. The COVID-19 Tracker leverages API-nCoV2019, providing up to date data from the WHO, CDC, among other sources.
